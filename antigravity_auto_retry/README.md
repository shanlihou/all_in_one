# Antigravity Auto Retry - 自动化屏幕匹配点击工具

这是一个基于 Python 的轻量级自动化工具，能够定期截取屏幕并自动识别、点击指定的目标图片。适用于自动化重复点击任务、游戏辅助或简单的流程自动化。

## 🚀 核心功能

- **实时监控**：每秒自动截屏并扫描屏幕内容。
- **多目标识别**：支持同时监控 `target_img` 文件夹下的多个目标模板（.png, .jpg）。
- **精准点击**：识别成功后，自动计算目标中心点坐标并执行鼠标左键点击。
- **全局快捷键控制**：
    - **F12**：随时随地一键停止脚本，无需切换回终端。
    - **紧急制动 (Fail-Safe)**：将鼠标快速移动到屏幕**任意一个角落**即可强制终止程序。
- **智能频率控制**：内置时间补偿机制，确保在处理耗时波动时仍能维持约 1 次/秒的检测频率。

## 🛠️ 环境要求

- Python 3.x
- 所需依赖库：
    - `opencv-python` (图像识别核心)
    - `pyautogui` (截图与鼠标控制)
    - `keyboard` (全局快捷键监测)
    - `numpy` (数据处理)

## 📦 安装步骤

1. **克隆/下载项目**到本地。
2. **安装依赖**：
   在项目根目录下打开终端，运行：
   ```bash
   pip install opencv-python pyautogui keyboard numpy
   ```

## 🖥️ 使用方法

1. **准备模板图片**：
   - 在项目根目录下的 `target_img` 文件夹中，放入你想要点击的目标图片。
   - 建议截图尽量精简且具有代表性，以提高匹配准确度。
   - 支持 `.png` 和 `.jpg` 格式。

2. **启动脚本**：
   运行以下命令开始监控：
   ```bash
   python main.py
   ```

3. **停止运行**：
   - 按下键盘上的 **F12** 键。
   - 或者直接将鼠标甩向屏幕角落触发紧急停止。

## ⚙️ 配置说明

你可以在 `main.py` 的开头部分根据需要调整以下参数：

- `MATCH_THRESHOLD` (默认 0.8)：匹配相似度阈值。调高可以减少误触，调低可以提高模糊匹配能力。
- `STOP_KEY` (默认 "f12")：更改停止脚本的全局快捷键。

## ⚠️ 注意事项

- **屏幕缩放**：如果你的系统开启了高 DPI 缩放（如 150%），请确保截图时和运行时的缩放比例一致，否则坐标可能会偏移。
- **权限说明**：在 Windows 上运行 `keyboard` 库检测全局快捷键时，建议以**管理员身份**运行终端，以确保快捷键捕获的稳定性。
- **安全提醒**：请确保 `target_img` 中不包含会导致死循环点击的图片（例如脚本启动后的控制台窗口本身）。
